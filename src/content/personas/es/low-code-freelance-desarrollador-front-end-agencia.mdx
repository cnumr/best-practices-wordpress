# Low-code → Freelance y Desarrollador Front-End de agencias

### Perfil:

- Necesitas crear un sitio web (o un sitio de comercio electrónico) para ti o para uno de tus clientes y tú o él/ella son conscientes de que el impacto digital no es nulo ([4% de las emisiones globales de GEI](https://www.greenit.fr/etude-empreinte-environnementale-du-numerique-mondial/)) ;
- Necesitas entregar rápidamente mientras mantienes los impactos ambientales bajo control tanto como sea posible ;
- Necesitas la información correcta rápidamente y aplicable sin desviarte de tu presupuesto ;
- Quieres conocer todos los puntos a verificar, desde el diseño hasta la entrega ;
- Sin embargo, no eres desarrollador, por lo que necesitas información adaptada y comprensible.

### Te identificas con estas afirmaciones:

> **LOW-CODE** → Tienen conocimientos básicos de HTML y CSS, utilizan un poco de código PHP como parche o para vincular plugins. Recurren a un tema bonito (o uno que harán bonito) y responden a sus necesidades o las necesidades de su cliente (o incluso incorporando más de las necesidades identificadas) y "cientos" de plugins y a menudo un constructor de páginas.

### Si es el caso:

Perfecto, estás en el lugar correcto. Cada ficha te indica y explica un problema y cómo resolverlo con una solución adaptada a ti.

### Si no es el caso:

No te preocupes, habrá una solución para cada perfil. ¡Hay una para todos!
